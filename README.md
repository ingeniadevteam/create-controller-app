## Using npx

```bash
$ npx @clysema/create-controller-app app_name [module1, module2, ...]
```

## Install globally and run

Install:
```bash
$ npm install --global @clysema/create-controller-app
```

Run:
```bash
create-controller-app app_name [module1, module2, ...]
```
